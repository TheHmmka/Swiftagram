Enumeration Cases

case `default`

Defaults to Device.default.browserUserAgent.

iOS 

case iOS(version: String)

Tied to a specific iOS version, e.g. 13_4_1.

You won't be able to use encrypted endpoints (e.g. Endpoint.Friendship.follow).

custom 

case custom(: String)

An entirely custom user agent.

You won't be abled to use encrypted endpoints (e.g. Endpoint.Friendship.follow), unless the user agent is of an Android device.

current 

case current

Tied to the current iOS version.

You won't be able to use encrypted endpoints (e.g. Endpoint.Friendship.follow).
