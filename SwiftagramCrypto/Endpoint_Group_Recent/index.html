stories 

var stories: Endpoint.Single<TrayItem.Collection, Error>

Recent stories for users and tags followed by the current user.

Keep in mind actual media might not be populated. Please rely on Endpoint.User(_:).stories or Endpoint.stories(_:), when dealing with more than one user at the time, to return the actual content.
